# 專題提案

## 專題名稱

智慧海象船班搭乘指南與停航通知 App

## 專題動機

離島或港口旅客在搭船前，常需要同時查詢天氣、海象與船班資訊。若資料分散在不同平台，旅客可能難以及時判斷是否適合搭乘。本專題希望透過單一 App 整合海象觀測與航港船班資料，提供更直覺的查詢與提醒介面。

## 目標使用者

- 搭乘渡輪或離島船班的旅客
- 需要查詢港口海況的人員
- 關注停航、延誤與船班異動的使用者

## 核心功能

- 顯示即時港口海象觀測
- 顯示即時船班狀態
- 根據海象與船班資料提供旅客搭乘提醒
- 支援船班搜尋與資料更新
- 支援淺色與深色模式

## 預期效益

本 App 可降低旅客查詢多個資料來源的成本，並透過風險提示協助使用者更快掌握是否適合前往港口或搭乘船班。

## 使用技術

- Android Studio
- Java / Kotlin
- XML Layout
- CWA 海象資料
- TDX 航港船班資料
