# 期末書面報告

## 專題名稱

智慧海象船班搭乘指南與停航通知 App

## 摘要

本專題開發一款 Android App，整合港口海象觀測與航港船班狀態，提供旅客即時查詢與搭乘風險提醒。使用者可透過 App 查看海況、船班資訊，並依據系統提示判斷是否適合前往港口或搭船。

## 系統功能

- 即時海象資料顯示
- 即時船班狀態顯示
- 船班搜尋
- 旅客搭乘提醒
- 淺色 / 深色模式切換

## 系統架構

Android App 主要程式碼位於 `app/` 目錄。畫面使用 XML Layout 建立，主流程由 `MainActivity.java` 控制，並透過 API Service 類別取得海象與船班資料。

## 使用資料來源

- 中央氣象署海象觀測資料
- TDX 運輸資料流通服務

## 結論

本 App 透過整合分散資料來源，讓旅客能更快速掌握海況與船班狀態，提升搭船前的資訊透明度與決策效率。
